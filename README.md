# Elementum Landing Page

A responsive React landing page for Elementum, built with Vite and plain CSS. The design uses a minimal white editorial layout, Gerbil typography, circular image masks, decorative blobs, floating badges, and premium Framer Motion animations.

## Features

- Responsive desktop, tablet, and mobile layouts
- Component-based React structure
- Plain CSS with custom properties
- Custom Gerbil font loaded locally
- Framer Motion text reveals, scroll reveals, floating avatars, hover motion, and subtle parallax
- Sections for Navbar, Hero, About, Services, Testimonials, Newsletter, and Footer
- Vercel-ready Vite build setup

## Tech Stack

- React
- Vite
- Plain CSS
- Framer Motion
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deploying To Vercel

1. Push the project to GitHub.
2. Open Vercel and import the GitHub repository.
3. Keep the framework preset as `Vite`.
4. Use `npm run build` as the build command.
5. Use `dist` as the output directory.
6. Deploy.
